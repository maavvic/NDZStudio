Generate fully functional WordPress plugins using AI directly from your WordPress dashboard.

== Description ==

AI Studio Generator is a revolutionary tool that allows you to create WordPress plugins by simply describing them in plain English. Powered by AI, it handles the coding, structure, and initial testing, allowing you to focus on your ideas.

**Key Features:**

* **Natural Language Generation:** Describe your plugin requirements, and the AI will generate the code.
* **Instant Installation:** Generate and install plugins with a single click.
* **Version History:** Roll back to previous versions of your generated plugins if needed.
* **Modification System:** Use AI to refine and add features to your existing generated plugins.
* **Safety Scans:** Automatically scans generated code for common WordPress security patterns.

**Note:** This plugin requires an API key from [NoDevZone AI Hub](https://app.nodevzone.com). You can start for free with a limited number of tokens.

== Installation ==

1. Upload the `wp-plugin-ai-generator` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the 'AI Generator' menu in your dashboard.
4. Enter your License Key from [app.nodevzone.com](https://app.nodevzone.com).

== Frequently Asked Questions ==

= Do I need an API key? =
Yes, you need a license key from app.nodevzone.com to connect to the AI generation services.

= Is the code safe? =
While the AI follows WordPress best practices and we include a security scanner, we always recommend reviewing any generated code before using it on a live production site.

= Can I modify the generated plugins? =
Yes! You can use the "Modify" feature to send instructions to the AI to add new features or fix issues in your generated plugins.
